"use client" component. Per D-12 through D-15 and UI spec sections 5 and 6.

```typescript
interface StructuredChoiceCardProps {
  data: StructuredPillData;
  onSend: (text: string) => void;

}
```

State:
- `selectedOptions: Set<string>` (for multi) or `selectedOption: string | null` (for single)
- `freeformText: string`
- `currentPage: number` (for multi pagination, 6 items per page)
- `visible: boolean` (starts true, false after save triggers fade-out)

Container: `rounded-xl border border-[#dae1ed] bg-white p-5 shadow-sm`. Wrap in a div with `transition-opacity duration-150` and `opacity-0` when `!visible`. After opacity transition ends and visible is false, call cleanup (parent handles removal).

Layout (top to bottom):
1. **Question header**: `<p className="text-sm font-semibold text-[#0f1114]">{data.question}</p>`

2. **Pagination** (multichoice only, when options.length > 6): Top-right of header. Show `{currentPage}/{totalPages}|{totalItems}` in `text-xs text-[#5b6780]`. ChevronLeft/ChevronRight buttons (from lucide-react, 16px). Disabled arrows: `opacity-30`. Active arrows: `text-[#5b6780] hover:text-[#0056d2]`.

3. **Options list** (`mt-3 space-y-1`):
   - **Multichoice**: Each option row: `flex items-center gap-3 py-2 px-3 rounded-lg hover:bg-[#f2f5fa] cursor-pointer`. Click toggles selection.
     - Checkbox: `w-[18px] h-[18px] rounded border-2 border-[#dae1ed] flex items-center justify-center`. Checked: `bg-[#0056d2] border-[#0056d2]` with white checkmark SVG (simple path).
     - Label: `text-sm text-[#0f1114]`.
   - **Single-choice**: Each option row: same layout. Click selects (deselects others).
     - Number badge: `w-6 h-6 rounded-full bg-[#e3eeff] text-[#0056d2] text-xs font-semibold flex items-center justify-center`. Numbers 1, 2, 3...
     - Selected: badge `bg-[#0056d2] text-white`, row `bg-[#f2f5fa]`.
     - Label: `text-sm text-[#0f1114]`.

4. **Freeform input** (`mt-3`): `<input>` with placeholder "Another {contextual}" for multi or "Something else" for single. Styles: `text-sm w-full border border-[#dae1ed] rounded-lg px-3 py-2 focus:outline-none focus:ring-2 focus:ring-[#0056d2]/20`.

5. **Action row** (`flex justify-between items-center mt-4`):
   - Left: "Skip For Now" — `<span className="text-sm text-[#5b6780] cursor-default">Skip For Now</span>`. Per D-16, non-functional (no onClick).
   - Right: "Save" button — `<button className="bg-[#0056d2] text-white rounded-full px-5 py-2 text-sm font-semibold hover:bg-[#0048b0] transition-colors">Save</button>`. Disabled when nothing selected and no freeform text.

- Collect all selected option labels. If freeform text is non-empty, append it.

- Set `visible = false` to trigger fade-out.
- After 150ms (transition duration), component should be removed. Use `onTransitionEnd` to signal parent, or just let parent handle via state.
