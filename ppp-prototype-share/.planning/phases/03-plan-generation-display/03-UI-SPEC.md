## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon-to-text gaps, inline padding within badges |
| sm | 8px | Course card internal gaps, badge padding, compact element spacing |
| md | 16px | Default element spacing, card padding, milestone header padding |
| lg | 24px | Section padding within plan view, milestone section gaps |
| xl | 32px | Gap between plan header and first milestone, layout column gaps |
| 2xl | 48px | Top/bottom padding of full plan view container |
| 3xl | 64px | Not used in this phase |

Exceptions:
- Course card thumbnails: 64x64px (maintains existing LIHP card pattern from `course-card.tsx`)
- Full plan view course thumbnails: 80x80px (larger for full plan view, per Figma)
- Chat panel: fixed 400px width (established in Phase 2, decision D-10)
- Plan panel horizontal padding: 32px (px-8, matching existing LIHP `lihp-page.tsx`)

## Interaction Contracts

### Plan Generation Flow (D-01, D-03)

1. **Trigger:** AI determines `ready_for_plan` is true, begins tool calling to search courses.
2. **During generation:** LIHP banner area shows `PlanLoadingSkeleton` (pulsing placeholder). Chat shows AI "thinking" indicator. Phase: `plan_generating`.
3. **On complete:** Full plan data arrives via tool call results. Banner swaps to populated `LearningPlanBanner` with real data. Chat shows "Learning plan created" confirmation message. Phase: `plan_generated`.
4. **Reveal:** All-at-once reveal (no progressive streaming of plan). Plan banner animates in with `opacity` + `translateY` transition (200ms ease-out).

### LIHP to Full Plan View (D-09)

1. **Trigger:** User clicks "View full plan" link on `LearningPlanBanner`.
2. **Transition:** Left panel content swaps from LIHP sections to `PlanView`. No route change (state swap in `LihpPage`). Chat panel remains at 400px on right. Phase: `viewing_plan`.
3. **Back:** "Back to home" link at top of `PlanView`. Returns left panel to LIHP sections. Phase returns to `plan_generated`.
4. **Animation:** Crossfade transition (150ms) between LIHP content and plan view. No layout shift -- same container dimensions.

### Course Card Interaction (D-11, D-12, D-13)

1. **Click course name:** Opens Coursera XDP page (`course.url`) in new tab via `target="_blank" rel="noopener noreferrer"`.
2. **Chevron:** Static decorative element or omitted. No expand/collapse.
3. **Hover:** Course name underlines on hover (`hover:underline`). No other hover effects on card.

### "Start learning plan" CTA (Claude's Discretion)

1. **Click:** Non-functional for prototype. Show a brief toast or subtle visual feedback ("Coming soon" tooltip or no-op). Do not navigate anywhere.

## Copywriting Contract

| Element | Copy |
|---------|------|
| Primary CTA (plan banner) | "Start Plan" (existing) with right chevron |
| Primary CTA (full plan view) | "Start learning plan" with right chevron |
| View full plan link | "View full plan" |
| Back to home link | "Back to home" (with left arrow) |
| Plan banner header label | "Your learning plan" (with sparkle icon) |
| Plan summary format | "{Role} . {skill1}, {skill2}, {skill3} . {duration range} . ~{N} hours/week" |

| Empty state (no plan yet) | Banner is hidden entirely. No empty state message needed -- the LIHP shows its normal content (trending, skills, collections). |
| Error state (plan generation fails) | Chat message: "I ran into an issue creating your plan. Let me try again." with automatic retry or "Try again" button in chat. No error shown in plan area -- banner stays hidden. |
| Chat confirmation (plan ready) | "Learning plan created" (posted by AI in chat after successful generation) |
| Milestone badge labels | "Core Track", "Professional Certificate" (hardcoded per milestone type, per Figma) |
| Activity badge labels | "Coding exercise", "Role play", "Guided Project", "Capstone Project" (hardcoded/mocked per D-11) |
| Start CTA tooltip (non-functional) | No tooltip -- button appears clickable but performs no action for prototype |

## State Machine Extension

Existing `AppPhase`:
```
"entry" | "chatting" | "ready_for_plan"
```

Extended for Phase 3:
```
"entry" | "chatting" | "ready_for_plan" | "plan_generating" | "plan_generated" | "viewing_plan"
```

| Phase | Left Panel | Right Panel | Banner State |
|-------|-----------|-------------|-------------|
| `entry` | Entry screen (full width) | -- | -- |
| `chatting` | LIHP sections | Chat at 400px | Hidden |
| `ready_for_plan` | LIHP sections | Chat at 400px | Hidden |
| `plan_generating` | LIHP sections | Chat at 400px (AI thinking) | Loading skeleton |
| `plan_generated` | LIHP sections | Chat at 400px | Populated with plan data |
| `viewing_plan` | PlanView (full plan) | Chat at 400px | N/A (user is in full plan view) |

```
+-------------------------------------------------------+----------+
| Back to home                                          |          |

| | +-----------------------------------------------+  | |          |
| | | [thumb] Course Name                       [>]  |  | |          |
| | |         skills line                            |  | |          |
| | |         partner . type . modules               |  | |          |
| | |         [badge] [badge]                        |  | |          |
| | +-----------------------------------------------+  | |          |
| | | [thumb] Course Name                       [>]  |  | |          |
| | +-----------------------------------------------+  | |          |
| +---------------------------------------------------+ |          |
|                                                       |          |
| +---------------------------------------------------+ |          |
| | Milestone 2: Core Skills                           | |          |
| | ...                                                | |          |
| +---------------------------------------------------+ |          |
+-------------------------------------------------------+----------+
```

- Plan panel: `flex-1` (takes remaining width after 400px chat)
- Plan content max-width: 860px, centered (`mx-auto`)
- Plan container padding: 32px horizontal (px-8), 24px vertical (py-6) -- matching existing LIHP
- Chat panel: fixed 400px, full height, border-left `border-[#dae1ed]`
- Milestone sections: stacked with 24px gap between them
- Course cards within milestone: separated by `divide-y divide-[#dae1ed]`

### LIHP Banner (within LIHP view)

- Same layout as existing `learning-plan-banner.tsx` but with dynamic data
- Rounded-xl container, border `#dae1ed`, white bg, 24px padding (p-6)
- Milestones listed in bordered rows within the banner
